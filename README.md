# Documents Validation - Permit System

Sistema de validação de documentos de transporte para Moçambique.

## 📋 Descrição

Este repositório contém a página de validação de Licenças de Transporte (Permits) emitidas pelo Ministério dos Transportes e Logística de Moçambique.

## 🚀 Como Usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   cd seu-repositorio
   ```

2. **Abra localmente:**
   - Abra o arquivo `index.html` no seu navegador
   - Ou use um servidor local (Python, Node.js, Live Server, etc.)

3. **Hospede no GitHub Pages:**
   - Vá para Settings do repositório
   - Navegue até Pages
   - Selecione a branch `main` como fonte
   - A página será publicada em: `https://seu-usuario.github.io/seu-repositorio/`

## 📁 Estrutura do Projeto

```
.
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── images/             # Pasta para imagens
│   └── emblema.png     # Logo do ministério

```
